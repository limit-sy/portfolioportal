LuminousConnect

「光」をテーマに、ゴルフの操作システムをパズルへと昇華させた3Dアクションゲームです。
光の弾を「打つ・狙う」という直感的な操作で、離れたオブジェクト同士を接続し、ステージのギミックを解き明かしていきます。
技術面では、LineRendererとRaycastによる軌道予測と接続判定の実装に加え、物語への没入感を高める独自の会話イベントシステムを構築。Unity 6の演出機能を活用し、遊びやすさと世界観の両立を追求しました。

  • 使用言語：C#
  • 開発環境：Unity Editor 6000.3.5.f2
  • 主要技術：LineRendererによる軌道予測 / 独自会話システム実装 / Raycast判定
  • 制作期間：1ヶ月

Animal_United_Front

5人のチーム開発で制作した、2D横スクロール・アクションゲームです。私はボスキャラクターのAIアルゴリズム全般を担当しました。
コルーチンとフラグ管理を組み合わせ、状況に応じた「逃亡」と「急襲」のステート制御を実装。Vector3.Lerpを用いた滑らかな移動や、Mathf.Atan2等の三角関数を活用し、プレイヤーの位置を正確に追従・攻撃するロジックを構築しました。
共 共同開発を通じ、GitHubでのコード管理や、チーム内の仕様（担当区分）に基づいたモジュール実装の実践的なスキルを習得しました。

  • 使用言語：C#
  • 開発環境：Unity Editor 6000.3.5.f2
  • 主要技術：コルーチンによる非同期制御 / 三角関数を用いた座標計算 / Lerpによる補間移動
  • 制作期間：1ヶ月

RunAndGunSurvivor

自動で前方に疾走するロボットを操作する3Dランアクションゲームです。
射撃と近接攻撃を使い分け、多彩なアイテムで自機を強化しながらゴールの踏破を目指します。
入力システムを工夫し、スティックの長押し時に『タメ』を作るリピート処理を実装。これにより、意図しない連続移動を防ぎつつ、スピーディーな操作感を実現しました。

  • 使用言語：C#
  • 開発環境：Unity Editor 6000.3.5.f2
  • 制作期間：1ヶ月

2D横スクロールシューティング

Scratchで作成した横スクロールシューティングアクションです。
